# mcp-wikipedia

MCP server to give client the ability to access Wikipedia pages

# Usage

```json
{
  "mcpServers": {
    "wikipedia": {
      "command": "uv",
      "args": [
        "--directory",
        "%USERPROFILE%/Documents/GitHub/mcp-wikipedia",
        "run",
        "python",
        "main.py"
      ]
    }
  }
}
```
